# DubaiPro Project

## Stack
- Next.js App Router
- TypeScript
- Prisma + PostgreSQL
- TailwindCSS
- next-intl

## Main Features
- B2B/B2C marketplace
- Product management
- Dynamic filters
- RFQ system
- Multi-language RTL support

## Architecture
- App Router structure
- Server actions preferred
- Prisma ORM
- Modular components

## Important Decisions
- Prisma instead of Strapi
- Minimal API routes
- RTL-first UI

## Current Progress

- Filters completed
- Import system in progress
